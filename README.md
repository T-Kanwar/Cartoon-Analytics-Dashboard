## 📊 Key Insights & Analytics Focus

This dashboard was built to answer critical business and creative questions regarding cartoon performance:
*   **Audience Engagement & Trends:** Analyzing how viewership numbers have shifted across different eras/networks (e.g., Cartoon Network, Nickelodeon, Disney).
*   **Rating & Sentiment Analysis:** Cross-referencing IMDB/Rotten Tomatoes scores against target demographics to find what truly makes a successful animated show.
*   **Character Metrics:** Tracking character screen time, popularity polls, and recurring tropes using custom visual distributions.

---

## 🛠️ Features & Technical Highlights

*   **Custom UI/UX & Visual Depth:** Designed with advanced presentation principles, featuring distinct container layering, smooth spacing, and **3D container effects** to give the dashboard a modern, sleek app-like feel.
*   **Dynamic KPI Cards:** Utilizes calculated fields to display real-time metric shifts (e.g., Average Rating, Total Viewers, Peak Eras) that automatically update based on global filters.
*   **Advanced Filtering:** Integrated interactive parameter controls and action filters allowing users to slice data by Network, Year, Genre, or Target Age Group seamlessly.
*   **Data Density & Layout:** Structured carefully to maximize information delivery without cluttering the screen, ensuring clean alignment and readable typography.
